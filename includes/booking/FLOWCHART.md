## 🔌 بحث AJAX

### ۱. دریافت خدمات دکتر

```
┌─────────────────────────────┐
│ Frontend: انتخاب دکتر       │
└────────────┬────────────────┘
             │
             ▼
      jQuery.ajax({
        action: 'mr_get_doctor_services',
        doctor_id: 5
      })
             │
             ▼
┌─────────────────────────────┐
│ Backend: mr_handle_get_...  │
│ • دریافت تنظیمات دکتر      │
│ • برگشت JSON               │
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│ Frontend: به‌روزرسانی form  │
│ • نمایش خدمات              │
│ • به‌روزرسانی قیمت‌ها       │
└─────────────────────────────┘
```

### ۲. ذخیره تنظیمات دکتر

```
┌──────────────────────────────┐
│ Frontend: submit form        │
└────────────┬─────────────────┘
             │
             ▼
      jQuery.ajax({
        action: 'mr_save_doctor_booking_settings',
        form_data: {...}
      })
             │
             ▼
┌──────────────────────────────┐
│ Backend: mr_handle_save...   │
```

## 🔐 Validation Pipeline

```
POST /wp-admin/admin-ajax.php
         │
         ▼
    Check nonce
         │
    ├─ ✓ Valid → Continue
    └─ ✗ Invalid → Error
         │
         ▼
    Check user role
         │
    ├─ ✓ Authorized → Continue
    └─ ✗ Unauthorized → Error
         │
         ▼
    Sanitize inputs
    (sanitize_text_field,
     intval, etc.)
         │
         ▼
    Validate data
    (check ranges, dates, etc.)
         │
    ├─ ✓ Valid → Process
    └─ ✗ Invalid → Error
         │
         ▼
    Update database
    (update_user_meta)
         │
         ▼
    Return JSON response
```
